# AB Switch

Provide a remote control GUI for OBS which behaves like a traditional A/B Video Switcher. 

[x] Establish a WebSocket connection with an OBS instance. 

[x] Enumerate OBS Sources as Buttons on the interface.

[x] When in OBS's Studio Mode, the Preview Source will be bordered by green, and the Program Source will be bordered by red.

[x] When in Studio Mode, tapping a Source's button will bring that Source to Preview, and tapping the 'GO' Button will transition the Preview Source to Program, while bringing the Program Source to Preview.

[] When not in Studio Mode, the Program Source will be bordered by red. 

[] When not in Studio Mode, tapping a Source's button will transition that Source to Program. The 'GO' Button is disabled, or hidden. 

[x] Provide Buttons to start and stop Streaming and to start and stop Recording.

[] Provide functionality with Downstream Keyer plugin and add DSK buttons.
